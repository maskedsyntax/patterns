# Rezumate App Store Screenshot Set

## Format

- Device class: 6.7-inch iPhone
- Export size: 1290 x 2796 px
- Template: large editorial headline, short support line, one iPhone mockup, cream background, bold black typography, thick black outlines, yellow CTA accents, blue app icon/accent.

## Screenshot Order

1. **Tailor your resume before you apply**  
   Supporting line: Check ATS-style fit against any job post.  
   Screen: Analyze flow with resume input, job description input, import/paste actions, and a private/local note.

2. **See what your resume is missing**  
   Supporting line: Score, keywords, gaps, and weak sections in one view.  
   Screen: Results view with an ATS-style score, component scores, missing keywords, and diagnosis cards.

3. **Improve weak bullets instantly**  
   Supporting line: Add stronger action, impact, and job keywords.  
   Screen: Improve state with before/after bullet treatment, highlighted stronger bullet, updated score, and save action.

4. **Match the role's keywords**  
   Supporting line: Find missing skills, tools, and role terms.  
   Screen: Keyword insights with matched chips, missing role terms, and suggested resume placement.

5. **Private by design**  
   Supporting line: Optimize locally. No account required.  
   Screen: Profile/privacy screen with on-device analysis, no-account, and local saved-history cards.

6. **Save variants. Export clean PDFs.**  
   Supporting line: One-time Pro unlock. No subscription.  
   Screen: Saved variants list with PDF export and Pro value cards.

## Files

- Editable source: `index.html`
- Export script: `render.js`
- Final PNGs: `exports/`
- Review contact sheet: `contact-sheet.jpg`

To regenerate, run:

```sh
node app-store/rezumate-screenshots/render.js
```
